# ngx-library

Simple extension library for [AngularJS] used in LMC projects. Mostly UI components, inspired by [Angular-UI] project.


## Prerequisites

- [Node.js]
- [NPM]


## Installation

````bash
npm install -g grunt

# in your local ngx folder
npm install
grunt
````

... and try demos
